jQuery Modal Dialog

这是一个最好用的轻量级的模态对话框. – 源码在 GitHub »

  • 自动绑定使用HTML语义
  • 没有图片& 轻量级 (文件只有1KB)
  • 非常简单的CSS样式定义
  • 使用自定义行为,参考jQuery events
  • 支持使用ESC键关闭对话框
  • 支持大多数浏览器

Example 1: 打开 & 关闭<a>连接

  1. 在文档中嵌入模态对话框HTML
  2. 创建一个超链接，并设置 rel="dialog:open" 和 href 属性为模态对话框的ID.
<!DOCTYPE html>
<html>
<head>
</style>
  <!-- Don't forget to include jQuery ;) -->
  <script src="jquery.modal.dialog.js" type="text/javascript" charset="utf-8"></script>
</head>
<body>

  <!-- Modal HTML embedded directly into document -->
  <div id="ex1" style="display:none;">
    <p>Thanks for clicking.  That felt good.  <a href="#" rel="dialog:close">Close</a> or press ESC</p>
  </div>

  <!-- Link to open the modal dialog -->
  <p><a href="#ex1" rel="dialog:open">Open Modal Dialog</a></p>

</body>
</html>

Example 4: AJAX

This first example uses rel="dialog:open" to automatically load the page contents into a modal dialog via AJAX:

<a href="ajax.html" rel="dialog:open">example</a>

This second example 演示如何手动加载Ajax页面为模态对话框. 注意Ajax响应必须被包裹在一个div类 dialog中:

<!-- Normal link -->
<a href="ajax.html" id="manual-ajax">second example</a>
//在AJAX回掉时打开模态窗口
$('#manual-ajax').click(function(event) {
  event.preventDefault();
  $.get(this.href, function(html) {
    $(html).appendTo('body').dialog();
  });
});
<!-- AJAX response must be wrapped in the modal's root class. -->
<div class="dialog">
  <p>Second AJAX Example!</p>
</div>

If you want more spinner styles, check out ajaxload.info.

Example 5: 禁用默认关闭窗口

This example 演示如何禁用关闭模态的默认方法： 

 $("#sticky").dialog({
  escapeClose: false,
  clickClose: false,
  showClose: false
});

If you do this, be sure to provide the user with an alternate method of closing the window.

Example 7: 渐变过渡

This example 演示如何通过指定的方式来做一个简单的渐变 fadeDuration 选项.

 $("#fade").dialog({
  fadeDuration: 100
});

You can also use fadeDelay to control the point during the overlay's fade in at which the modal fades in. For example, to fade in the modal when the overlay transition is 50% complete:

 $("#fade").dialog({
  fadeDuration: 1000,
  fadeDelay: 0.50
});

The default value is 1.0, meaning the window transition begins once the overlay transition has finished. Values greater than 1.0 mean there is a delay between the completed overlay transition and the start of the window transition, example:

 $("#fade").dialog({
  fadeDuration: 1000,
  fadeDelay: 1.75 // 覆盖结束后,将在750毫秒淡出..
});

Tip: set fadeDelay: 0 覆盖和窗口褪色的时间.

In the spirit of keeping this library small, fading is the only supported transition. When the modal is closed, both the overal and window transition out simultaneously.

Example 10: 为关闭按钮自定义CSS

This example demonstrates how to add extra classes to the close button (for custom styles for the close button): 

 $("#custom-close").dialog({
  closeClass: 'icon-remove',
  closeText: '!'
});
